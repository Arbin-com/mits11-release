MITS11 installation

One-line install. Default is latest stable version. Use alpha or a version to override.

PowerShell 5+
Stable (default)
irm https://arbin-com.github.io/mits11-release/install.ps1 | iex
Alpha
irm https://arbin-com.github.io/mits11-release/install.ps1 | iex alpha
Specific version
irm https://arbin-com.github.io/mits11-release/install.ps1 | iex 5.0.1
Debian-based distros
Stable (default)
curl -fsSL https://arbin-com.github.io/mits11-release/install.sh | bash
Alpha
curl -fsSL https://arbin-com.github.io/mits11-release/install.sh | bash -s -- alpha
Specific version
curl -fsSL https://arbin-com.github.io/mits11-release/install.sh | bash -s -- 5.0.1
Terminal (bash/zsh)
Stable (default)
curl -fsSL https://arbin-com.github.io/mits11-release/install.sh | bash
Alpha
curl -fsSL https://arbin-com.github.io/mits11-release/install.sh | bash -s -- alpha
Specific version
curl -fsSL https://arbin-com.github.io/mits11-release/install.sh | bash -s -- 5.0.1
